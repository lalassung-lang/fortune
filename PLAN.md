## 3. 기능 상세 설계

### 3-1. 입력부

- **생년월일 입력**: `st.date_input()` — 달력 UI
- **성별 선택**: `st.radio()` — 남 / 여 (사주 해석 참고용)
- **버튼**: `st.button("✨ 오늘의 운세 보기")`

---

### 3-2. 사주 운세 (🧧)

- **계산 방법**: 생년월일 → 연주(年柱)·월주(月柱)·일주(日柱)·시주(時柱) 추출
- **OpenAI 프롬프트 방향**:
  > "오늘({today}) 기준, 생년월일 {birth}인 {gender}의 사주를 분석해줘.
  > 사주의 오행(木火土金水) 흐름과 오늘 하루의 운세를 재미있고 친근하게 설명해줘.
  > 연애운, 금전운, 건강운으로 나눠서 알려줘."
- **출력 형식**: 오행 요약 + 오늘의 사주 운세 (연애·금전·건강)

---

### 3-3. 별자리 운세 (♈)

- **계산 방법**: 생년월일 → 태양 별자리 자동 판별
  - 예) 3/21~4/19 → 양자리(Aries)
- **OpenAI 프롬프트 방향**:
  > "오늘({today}) {zodiac_sign}자리의 운세를 알려줘.
  > 전체운, 연애운, 직업운, 행운의 색상과 숫자도 포함해서
  > 재미있게 MZ 감성으로 작성해줘."
- **출력 형식**: 별자리 이름 + 오늘의 운세 + 행운 아이템

---

### 3-4. 띠 운세 (🐉)

- **계산 방법**: 출생 연도 → 12지신 띠 판별
  - 예) 2000년 → 용띠
- **OpenAI 프롬프트 방향**:
  > "오늘({today}) {zodiac_animal}띠의 운세를 알려줘.
  > 오늘 조심해야 할 것, 오늘 잘 풀리는 것, 궁합 좋은 띠도 알려줘.
  > 재치 있고 유머러스하게 써줘."
- **출력 형식**: 띠 이모지 + 오늘의 운세 + 오늘의 팁

---

### 3-5. 타로카드 (🎴)

#### GPT가 타로를 할 수 있나?
> **YES.** GPT-4o는 메이저·마이너 아르카나 78장의 상징, 정방향/역방향 의미를 모두 알고 있음.
> 카드 뽑기는 Python `random` 으로 처리하고, 해석만 GPT에 맡기면 됨.

#### 타로 카드 UI — 플립 애니메이션 구현 방식

- `st.components.v1.html()` 로 HTML/CSS/JS 블록을 Streamlit에 삽입
- CSS `perspective` + `rotateY` 트랜지션으로 카드 뒤집기 애니메이션 구현 (순수 CSS, 외부 라이브러리 불필요)
- 카드 앞면: 신비로운 뒷면 그래픽 (CSS로 보라/금색 패턴 생성)
- 카드 뒷면: 카드 이름 + 이모지 심볼

#### 플로우

```
[카드 3장 앞면이 보임 — 클릭 유도]
        ↓ 사용자가 카드 클릭
[선택한 카드 플립 애니메이션 (0.6s)]
        ↓
[카드 이름 공개 — 예: "The Moon 🌕"]
        ↓
[GPT 해석 스트리밍 출력]
 "오늘의 {카드명}은 당신에게 이런 의미입니다..."
```

#### OpenAI 프롬프트 방향
> "오늘 {birth}생 {gender}이 타로카드 {card_name}({upright_or_reversed})을 뽑았어.
> 오늘의 상황에 맞게 이 카드의 의미를 해석해줘.
> 사랑, 일, 조언 3가지 관점에서 재미있고 신비롭게 작성해줘.
> 마지막에 오늘의 타로 메시지 한 줄도 추가해줘."

### 3-6. 종합 리포트 + 추천 로또번호 (📋🍀)

- **생성 시점**: 사주·별자리·띠·타로 4가지 결과가 **모두** 준비된 뒤 자동 생성
- **포인트**: "모든 운명의 조각이 모였다" 연출 — 4종 운세가 다 나와야 잠금 해제

#### 파트 A — 종합 리포트

- **OpenAI 프롬프트 방향**:
  > "아래는 오늘 {birth}생 {gender}의 운세 분석 결과야.
  > - 사주운세: {saju_result}
  > - 별자리운세: {zodiac_result}
  > - 띠운세: {animal_result}
  > - 타로카드: {tarot_card_name}({정/역방향}) — {tarot_result}
  >
  > 이 네 가지를 모두 종합해서 다음 항목을 작성해줘:
  > 1. ⚠️ 오늘 조심해야 할 것 (2~3가지, 구체적으로)
  > 2. 🎁 오늘의 행운 아이템 (색상, 음식, 숫자, 방향 등 3가지)
  > 3. 💡 오늘 하루 이렇게 보내세요 (한 줄 실천 팁)
  > 4. 🎴 타로가 오늘 당신에게 전하는 핵심 메시지 (한 줄)
  > 5. 🌟 오늘의 한마디 (응원 멘트 또는 명언)
  > 재미있고 실용적으로 작성해줘."
- **출력 형식**: 5개 항목을 카드형 레이아웃으로 풀 width 표시

#### 파트 B — 추천 로또번호

- **생성 방법**: 종합 리포트와 같은 맥락(4종 운세 결과)을 바탕으로 로또번호 추천
- **OpenAI 프롬프트 방향**:
  > "위의 사주·별자리·띠·타로 운세 결과를 모두 바탕으로,
  > 오늘 이 사람에게 가장 어울리는 로또 번호 6개를 1~45 범위에서 추천해줘.
  > 번호마다 왜 이 번호를 골랐는지 짧고 재미있게 이유도 달아줘."
- **출력 형식**: 번호 6개를 색깔 볼 형태로 시각화 + 각 번호 선택 이유
- **위치**: 종합 리포트 바로 아래, 같은 섹션 내 하단 배치

## 4. OpenAI API 연동 설계

### API 키 설정 방법

프로젝트 루트에 `.env` 파일을 생성하고 아래처럼 입력하세요:

```
OPENAI_API_KEY=sk-여기에-본인-API키-붙여넣기
```

> ⚠️ `.env` 파일은 절대 GitHub에 올리지 마세요! `.gitignore`에 추가 필수.

### 사용 모델

| 용도 | 모델 | 이유 |
|------|------|------|
| 운세 생성 | `gpt-4o` | 맥락 이해 + 풍부한 문장 생성 |
| (저비용 대안) | `gpt-4o-mini` | 비용 절감 시 사용 가능 |

### API 호출 구조

```
사용자 입력 (생년월일 + 성별)
    ↓
생년월일 파싱 → 사주 천간지지 계산
                → 별자리 판별
                → 띠 판별
                → 타로 카드 랜덤 3장 뽑기 (random, 정/역방향 포함)
    ↓
[STEP 1] 개별 운세 3종 병렬 호출 → 섹션 1 상단 렌더링
  ① 사주운세 프롬프트
  ② 별자리운세 프롬프트
  ③ 띠운세 프롬프트
    ↓
[STEP 2] 타로 카드 UI 표시 → 사용자 카드 클릭 대기 → 섹션 1 하단 렌더링
  ④ 타로해석 프롬프트 (카드 클릭 시 on-demand, 스트리밍 출력)
    ↓
[STEP 3] ①②③④ 모두 준비되면 자동 생성 → 섹션 2 렌더링
  ⑤ 종합리포트 프롬프트 (사주+별자리+띠+타로 결과 전부 포함)
  ⑥ 로또번호 프롬프트   (⑤와 같은 맥락, 종합 섹션 하단 배치)
    ↓
결과 파싱 → Streamlit 화면 렌더링
  - ①②③④ → 섹션 1: 2×2 그리드 (타로는 클릭 인터랙션 포함)
  - ⑤⑥   → 섹션 2: 풀 width 종합 리포트 + 로또번호 볼 시각화
```

## 9. 타로 플립 게임 — 기술 구현 상세

### 핵심 기술: `st.components.v1.html()`

Streamlit은 HTML/CSS/JS 블록을 직접 삽입할 수 있음.
타로 카드 애니메이션은 **순수 CSS + 바닐라 JS**로 구현 (외부 라이브러리 없음).

### 카드 플립 CSS 핵심 원리

```css
.card { perspective: 1000px; }
.card-inner {
  transition: transform 0.6s;
  transform-style: preserve-3d;
}
.card.flipped .card-inner {
  transform: rotateY(180deg);  /* 클릭 시 뒤집힘 */
}
.card-front, .card-back {
  backface-visibility: hidden;  /* 뒷면 숨기기 */
}
.card-back { transform: rotateY(180deg); }
```

### 카드 선택 → GPT 해석 연동 흐름

```
[카드 클릭] → JS: postMessage(cardIndex) → Streamlit 컴포넌트 이벤트
    ↓
st.session_state에 선택 카드 저장
    ↓
openai.chat.completions.create(stream=True) 호출
    ↓
st.write_stream() 으로 해석 실시간 스트리밍 출력
```

### 재미 포인트

- 카드 선택 전: "운명의 카드를 고르세요... 🔮" 텍스트 흔들림 애니메이션
- 뒤집힐 때: 카드 주변 빛나는 파티클 효과 (CSS box-shadow animation)
- Death 카드가 나오면: "💀 무섭지 않아요! 사실 변화와 새 시작을 뜻해요" 해석 멘트 자동 포함
